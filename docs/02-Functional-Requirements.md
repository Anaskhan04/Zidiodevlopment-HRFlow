# Functional Requirements

## Authentication

- Users can log in using email and password.
- Users can securely log out.
- Users can reset forgotten passwords.
- Protected routes require authentication.
- Users remain logged in using refresh tokens.

---

## Organization Management

- Create organization
- Update organization profile
- Invite users
- View organization details

---

## Employee Management

HR Admin should be able to:

- Add employees
- Edit employee details
- View employee profile
- Search employees
- Filter employees
- Activate employee
- Deactivate employee
- Assign department
- Assign manager
- Upload employee documents

Employees should be able to:

- View their own profile
- Update limited personal information

---

## Department Management

HR Admin can:

- Create department
- Edit department
- Delete department
- Assign department head
- View department employees

---

## Attendance

Employees can:

- Check In
- Check Out

Managers can:

- View attendance

HR can:

- Export attendance
- Correct attendance
- Generate attendance reports

---

## Leave Management

Employees can:

- Apply leave
- Cancel leave
- View leave history

Managers can:

- Approve leave
- Reject leave

HR can:

- Configure leave types
- View organization leave reports

---

## Dashboard

Display

- Employee count
- Attendance summary
- Leave summary
- Department statistics
- Recent activities

---

## Notifications

- Leave approved
- Leave rejected
- Employee invited
- Employee activated
- Attendance reminders

---

## Role Based Access

Owner

- Full access

HR Admin

- HR operations

Manager

- Team management

Employee

- Personal dashboard
